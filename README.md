# Tribe-Retail-Store

### Project Description
TRIBE is a retail website for a fictional sportswear brand launched in 2007.The brand is inspired by the spirit of Ubuntu and focuses on eco-friendly, colourful sneakers and clothing for athletes and fitness-conscious people.This website was built as part of a first-year Web Development assignment for IIE Rosebank College.

**Slogan**: "We are the world"
**Mission**: To bring inspiration and innovation to every athlete regardless of how they look.
**Vision**: To do everything possible to expand human potential.

---

### Features
- **Home Page**: Showcases brand story, mission, and featured products
- **Services Page**: Displays TRIBE's eco-friendly sneakers and clothing  range
- **About Page**: Explains brand history and Ubuntu inspiration
- **Enquiry Page**: Enquiry details of clients
- **Contact Page**: Contact form and store location details
- **Responsive Design**:Works on desktop and mobile devices

---

### Technologies Used
**HTML5**: Website structure and content 
**GitHub Pages**: Hosting the live site
